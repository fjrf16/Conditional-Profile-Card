## INTRODUCTION

There is an example of using a interactive profile page with Javascript, HTML and CSS.


### Technologies
Project is created with:
* CSS
* HTML
* JAVASCRIPT
* BOOTSTRAP


### Setup
To run this project, install it locally using npm:

```
$ cd exercise-conditional-profile-card
$ npm install
$ npm run start
```
